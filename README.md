# To-Do List App

Projeto simples de lista de tarefas (To-Do List) desenvolvido com HTML, CSS e JavaScript.


# Funcionalidades

- Adicionar tarefas
- Remover tarefas
- Marcar tarefas como concluídas
- Persistência de dados com LocalStorage (as tarefas permanecem mesmo após atualizar a página)

# Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage
- Git e GitHub


# Como funciona

- O usuário digita uma tarefa no campo de input
- A tarefa é adicionada à lista
- Clique na tarefa remove ela
- As tarefas são salvas automaticamente no navegador

# Objetivo do projeto

Projeto desenvolvido para praticar:
- Manipulação do DOM
- Lógica com JavaScript
- Armazenamento local no navegador
- Versionamento com Git e GitHub
